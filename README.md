Тема курсової роботи: Навчальна лабораторія кафедри закладу вищої освіти (варіант №9)

Структура проєкту:

* **navlab-frontend/** — клієнтська частина додатку. Побудована на React, Vite та чистому CSS. Підтримує динамічну зміну тем (світла/темна) та інтерактивні CSS-анімації в галереї.
* **navlab-backend/** — серверна частина додатку. Реалізована на Node.js (Express) + SQLite. Забезпечує збереження даних та обробку запитів.

Інструкція з локального розгортання:
Для запуску проєкту на локальній машині переконайтеся, що у вас встановлено **Node.js**.

- Запуск бекенд-сервера (API)
Відкрийте термінал, перейдіть у папку сервера та запустіть його за допомогою Node:

cd navlab-backend

npm install

node server.js

- Запуск фронтенд-сервера:

cd navlab-frontend

npm install

npm run dev
